# Cppsarif: Tree View for C++ Compiler Output

Cppsarif is a vscode extension that organizes linear C++ compiler output into a clear tree view.

## Show

- Collapsed:
![collapse](share/collapse.png)

- Expanded:
![expand](share/expand.png)

- Click the tags on the right to jump directly to the corresponding code:
![skip](share/skip.png)

## Install

- You can install it from the VS Code Marketplace with the name `cppsarif`.
